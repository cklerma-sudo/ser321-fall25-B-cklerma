You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply?num1=INT&num2=INT -- multiplies num1 and num2 and returns the product (example: 18.220.199.157:9000/multiply?num1=2&num2=3)
  • /github?query=PATH -- pulls JSON from the GitHub REST API for the given path (example: 18.220.199.157:9000/github?query=users/amehlhase316/repos)
  • /printMultiString?string=STRING&amount=INT -- prints STRING repeated amount times (example: 18.220.199.157:9000/printMultiString?string=A&amount=5)
  • /randomNumber?min=INT&max=INT -- returns a random integer between min and max (example: 18.220.199.157:9000/randomNumber?min=0&max=100)
